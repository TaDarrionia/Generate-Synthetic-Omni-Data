# Synthetic Omni-Channel Retail Data Generator

This repository contains multiple synthetic data generators that simulate real-world omnichannel retail operations — from basic order generation to advanced parcel-level logistics simulation.

---

## Available Modules

### Baseline Generator (v1)

A lightweight synthetic dataset generator for:

- Order demand
- SKU-level sales
- Basic fulfillment (BOPIS vs Ship-to-Home)
- Simple shipping simulation

Located in: `/v1_baseline/`

---

### Advanced Simulation Engine (v2)

A high-fidelity simulation engine modeling:

- Customer behavior (Pareto distribution)
- Store network constraints
- Multi-package shipments
- Box packing logic
- Distance-based zone pricing
- Carrier rate cards (FedEx-style)
- Time-based operations (order → ship → delivery)

Located in: `/advanced_simulation/`
